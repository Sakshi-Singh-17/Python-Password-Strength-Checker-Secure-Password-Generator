# 🔐 Password Strength Checker & Generator (Python)

A Python-based security tool that evaluates the strength of a password and suggests improvements.  
If the password is weak, the program automatically generates a strong and secure password.

## 🚀 Features

- Password strength detection
- Password scoring system
- Checks for:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
  - Minimum length
- Suggests improvements for weak passwords
- Automatically generates a strong password
- Modular Python code structure

## 🛠 Technologies Used

- Python
- Random module
- String validation methods

## 📊 Password Strength Logic

The program evaluates password strength using:

- Character variety (uppercase, lowercase, numbers, symbols)
- Password length
- Complexity rules

It then calculates a **password strength score** and classifies the password as:

- Very Weak
- Weak
- Moderate
- Strong
- Very Strong
